== Description ==

The WordPress Business Directory plugin by CreativeMinds allows you to build a business directory on your WordPress site. This plugin makes it easy to start any kind of directory and embed it on your site. Using an Add-On, this plugin can also support the submission of new directory entries as well as editing of entries by external users who wish to be listed on your business directory.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define your business directory settings - for example how many buisnesses will show in the index page
4. Create  your first business page 

Note: You must have a call to wp_head() in your template in order for the footnote js and css to work properly.  If your theme does not support this you will need to link to these files manually in your theme (not recommended).
